# 🔴 ПРОБЛЕМЫ DESKTOP-ПРИЛОЖЕНИЯ (на 02.03.2026)

## Проблема 1: Палитра цветов не сохраняется между запусками

**Статус:** ✅ **ИСПРАВЛЕНО** (02.03.2026)

**Симптомы:**
- Пользователь выбирает цвета в "Сервис → Цвет интерфейса"
- Нажимает "Применить" — цвета применяются
- Закрывает приложение, запускает снова — цвета сбросились на белые/по умолчанию

**Причина:**
- `PALETTE_FILE = os.path.join("data", "palette.json")` использовал относительный путь
- При запуске .exe рабочая директория могла вычисляться неправильно

**Решение:**
В `Дерево/constants.py` изменено вычисление `PALETTE_FILE`:
```python
if getattr(sys, "frozen", False):
    # Запуск .exe: данные в папке data/ рядом с exe
    _base_dir = os.path.dirname(sys.executable)
else:
    # Запуск из исходников: данные в папке data/ проекта
    _base_dir = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
PALETTE_FILE = os.path.join(_base_dir, "data", "palette.json")
```

**Диагностика:**
В `main.py` добавлено логирование:
```python
print(f"[MAIN] PALETTE_FILE = {constants.PALETTE_FILE}")
print(f"[MAIN] Применено {len(_loaded_palette)} цветов из палитры")
```

**Файлы изменены:**
- `Дерево/constants.py` — абсолютный путь к `PALETTE_FILE`
- `main.py` — добавлено логирование загрузки палитры

## Проблема 3: Дерево стартует пустым, рисуется с задержкой

**Статус:** ✅ **ИСПРАВЛЕНО** (02.03.2026)

**Симптомы:**
- При запуске показывается окно приветствия
- Холст пустой (белый/серый)
- Через 2-5 секунд дерево появляется

**Причина:**
- `update_idletasks()` недостаточно для немедленной перерисовки

**Решение:**
В `Дерево/app.py` (`__init__()`) добавлена принудительная перерисовка:
```python
# Принудительно применяем цвета из палитры ПЕРЕД отрисовкой
self.apply_ui_colors_from_palette()

# Обновляем холст сразу после применения цветов
self.root.update_idletasks()
self.root.update()  # Принудительная перерисовка окна

self.refresh_view()
self.root.update_idletasks()  # Ещё раз обновляем после отрисовки
```

**Файлы изменены:**
- `Дерево/app.py` — добавлен `self.root.update()` для немедленной перерисовки

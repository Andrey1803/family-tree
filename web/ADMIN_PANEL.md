# 👑 Админ-панель веб-версии

## Обзор

Админ-панель предназначена для пользователя с логином **`admin`** и предоставляет возможность:
- Просматривать статистику системы
- Видеть всех зарегистрированных пользователей
- Просматривать деревья пользователей
- Переходить к деревьям пользователей для просмотра

## Доступ к админ-панели

### 1. Вход как администратор

На странице входа https://family-tree-production-0e7d.up.railway.app/login:

1. Нажмите кнопку **"👑 Администратор? Войти как admin"**
2. Введите пароль (по умолчанию `admin123` или ваш пароль)
3. Нажмите **"Войти"**

### 2. Переход в админ-панель

После успешного входа:
- В верхней панели появится кнопка **"👑 Админ"**
- Нажмите на неё для перехода в админ-панель
- Или перейдите напрямую: `https://family-tree-production-0e7d.up.railway.app/admin`

## Вкладки админ-панели

### 📊 Статистика

Отображает общую информацию о системе:
- **Пользователей** — общее количество и активных
- **Деревьев** — количество семейных деревьев
- **Персон** — общее количество записей о людях
- **Сессий** — активных сессий за последний час

Также показываются:
- График активности по дням (30 дней)
- Последние регистрации пользователей
- Последние синхронизации

### 👥 Пользователи

Таблица всех пользователей с возможностью:
- **Поиск** по логину и email
- **Просмотр деревьев** — переход к деревьям пользователя
- **Активировать/Деактивировать** — включение/отключение доступа

Поля таблицы:
- ID — идентификатор пользователя
- Логин — имя для входа
- Email — контактный email
- Регистрация — дата создания аккаунта
- Последний вход — дата последнего входа
- Статус — активен/не активен
- Админ — флаг администратора (👑)
- Действия — кнопки управления

### 🌳 Деревья

Список всех семейных деревьев:
- Название дерева
- Количество персон
- Количество браков
- Дата последнего обновления

Для каждого дерева доступен **просмотр деталей** — модальное окно со списком всех персон.

## API эндпоинты

Админ-панель использует следующие API:

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/api/admin/stats` | GET | Статистика системы |
| `/api/admin/users` | GET | Список пользователей |
| `/api/admin/user/<id>/toggle` | POST | Изменить статус |
| `/api/admin/user/<id>/trees` | GET | Деревья пользователя |
| `/api/admin/trees` | GET | Все деревья |

## Интеграция с сервером синхронизации

Админ-панель работает через сервер синхронизации Railway:
- https://ravishing-caring-production-3656.up.railway.app

Все данные о пользователях и деревьях хранятся в центральной базе данных.

## Безопасность

- Доступ только для пользователя `admin`
- Проверка токена сессии
- Интеграция с bcrypt для хеширования паролей

## Локальное тестирование

Для тестирования локально:

```bash
cd web
python app.py
```

Откройте http://127.0.0.1:5000/login и войдите как `admin`.

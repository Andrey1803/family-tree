### Веб-версия (`web/app.py`)

**Загрузка дерева:**
```python
# GET /api/tree
req = urllib.request.Request(
    f"{SYNC_SERVER_URL}/api/sync/download",
    headers={'Authorization': f'Bearer {server_token}'},
    method='GET'
)
```

**Выгрузка дерева:**
```python
# POST /api/tree
req = urllib.request.Request(
    f"{SYNC_SERVER_URL}/api/sync/upload",
    data=json.dumps({'tree': tree_data, 'tree_name': name}).encode(),
    headers={'Authorization': f'Bearer {server_token}'},
    method='POST'
)
```

### Оконная версия (`Дерево/sync_client.py`)

**Загрузка дерева:**
```python

    result = self._request('/api/sync/download')
    return result
```

## ⚠️ Возможные проблемы и решения

### 1. Ошибка 401 Unauthorized
**Причина:** Неверный логин/пароль или истёк токен

**Решение:**
- Проверьте учётные данные
- Выполните вход заново

### 2. Ошибка 404 Not Found
**Причина:** Дерево не найдено на сервере

**Решение:**
- Загрузите дерево на сервер впервые
- Проверьте подключение к серверу

### 3. Ошибка синхронизации
**Причина:** Конфликт данных между локальной и серверной версией

**Решение:**
- Используйте функцию "Синхронизировать" в меню
- Выберите приоритетную версию (локальная или сервер)

# 👑 Админ-панель для Андрея Емельянова

## Описание

**Полная админ-панель** — инструмент для управления всеми пользователями и их деревьями семейного древа.

### Возможности

✅ **Данные с сервера** — вся информация подтягивается онлайн  
✅ **Все пользователи** — полный список с деталями  
✅ **Все деревья** — просмотр и управление  
✅ **Блокировка** — управление доступом пользователей  
✅ **Статистика** — общая информация по системе  

---

## Как открыть

1. **Войдите** в программу под пользователем **Емельянов Андрей**
2. **Меню** → **Файл** → **👑 Админ-панель**
3. **Введите пароль** (если не сохранён)
4. **Панель загрузит данные** с сервера

## Интерфейс

### 📊 Статистика системы (верхняя панель)

| Показатель | Описание |
|------------|----------|
| 👥 Пользователей | Общее количество зарегистрированных пользователей |
| 🌳 Деревьев | Количество всех деревьев в системе |
| 📄 Персон | Общее количество персон во всех деревьях |
| ✅ Активных | Количество активных (незаблокированных) пользователей |

### 👥 Вкладка "Пользователи"

Таблица со всеми пользователями:

| Колонка | Описание |
|---------|----------|
| Логин | Имя пользователя |
| Email | Электронная почта |
| Создан | Дата регистрации |
| Деревья | Количество деревьев |

#### Контекстное меню (ПКМ по пользователю):

- **👁️ Просмотреть деревья** — показать деревья пользователя
- **🔓 Сбросить пароль** — отправить новый пароль
- **⚠️ Заблокировать/Разблокировать** — изменить статус
- **🗑️ Удалить пользователя** — удалить аккаунт и все деревья

### 🌳 Вкладка "Все деревья"

Таблица со всеми деревьями системы:

| Колонка | Описание |
|---------|----------|

| Название | Имя дерева |
| Обновлено | Дата последнего изменения |
| Размер | Размер файла |
| Персон | Количество персон в дереве |

#### Контекстное меню (ПКМ по дереву):

- **👁️ Просмотреть дерево** — открыть дерево для просмотра
- **📥 Скачать** — загрузить дерево на компьютер
- **🗑️ Удалить** — удалить дерево с сервера

## API Endpoints

Админ-панель использует следующие API:

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/api/admin/stats` | GET | Статистика системы |
| `/api/admin/users` | GET | Список всех пользователей |
| `/api/admin/trees` | GET | Список всех деревьев |
| `/api/admin/user/{id}/toggle` | POST | Блокировка/разблокировка |
| `/api/admin/user/{id}/reset-password` | POST | Сброс пароля |
| `/api/admin/user/{id}` | DELETE | Удаление пользователя |
| `/api/admin/trees/{id}` | DELETE | Удаление дерева |

## Безопасность

### Доступ

- ✅ Только для пользователя **Емельянов Андрей**
- ✅ Требуется ввод пароля
- ✅ Используется Basic Auth для API

⚠️ **Блокировка** — пользователь не сможет войти в систему  
⚠️ **Сброс пароля** — старый пароль перестаёт действовать

## Устранение проблем

### "Ошибка авторизации"
Проверьте логин/пароль администратора. По умолчанию: `admin` / `admin123`

### "Доступ запрещён"
Админ-панель доступна только для пользователя **Емельянов Андрей**

### "Ошибка подключения"
Сервер синхронизации недоступен. Проверьте интернет-соединение.

### "Не загружаются пользователи"
Возможно на сервере ещё нет пользователей или ошибка API.

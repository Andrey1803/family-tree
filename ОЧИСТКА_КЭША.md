# 🔄 ОЧИСТКА КЭША БРАУЗЕРА

## ❌ ПРОБЛЕМА

Тестовая страница работает ✅, но основное приложение выдаёт:
```
[SAVE] ❌ Network error: TypeError: Failed to fetch
```

**Причина:** Браузер использует **старую закэшированную версию** `tree.js`.

---

## ✅ РЕШЕНИЕ

### Способ 1: Полная очистка кэша (рекомендуется)

**Chrome/Edge:**
1. Нажмите `Ctrl + Shift + Delete`
2. Выберите "Кэшированные изображения и файлы"
3. Нажмите "Удалить"
4. Обновите страницу `Ctrl + F5`

**Firefox:**
1. Нажмите `Ctrl + Shift + Delete`
2. Выберите "Кэш"
3. Нажмите "Очистить"
4. Обновите страницу `Ctrl + F5`

---

### Способ 2: Режим инкогнито

**Chrome/Edge:** `Ctrl + Shift + N`
**Firefox:** `Ctrl + Shift + P`

Затем откройте:
```
https://family-tree-production-0e7d.up.railway.app
```

---

### Способ 3: Hard Reload

**Windows:**
- `Ctrl + Shift + R` или `Ctrl + F5`

**Mac:**
- `Cmd + Shift + R`

---

### Способ 4: Очистка через Developer Tools

1. Откройте DevTools (`F12`)
2. **ПКМ на кнопке обновления** (рядом с адресной строкой)
3. Выберите **"Empty Cache and Hard Reload"**

Или:
1. DevTools → вкладка **Network**
2. Поставьте галочку **"Disable cache"**
3. Обновите страницу

---

## 🔍 ПРОВЕРКА

После очистки кэша:

1. Откройте DevTools (`F12`)
2. Вкладка **Network**
3. Обновите страницу
4. Найдите `tree.js`
5. Проверьте что **размер не "disk cache"** и есть `(from service worker)` или `(memory cache)`

**Версия должна быть новая:**
```
tree.js?v=20260313-1  (или новее)
```

---

## 📊 ЕСЛИ ВСЁ ЕЩЁ НЕ РАБОТАЕТ

### Проверьте Service Workers

1. DevTools → вкладка **Application**
2. **Service Workers**
3. Нажмите **"Unregister"** если есть активные
4. Обновите страницу

### Проверьте расширения

Некоторые расширения блокируют запросы:
- **uBlock Origin**
- **AdBlock**
- **Privacy Badger**
- **NoScript**

**Отключите их** для этого сайта или добавьте в исключения:
```
family-tree-production-0e7d.up.railway.app
```

---

## 🎯 ПРОВЕРКА УСПЕХА

После очистки кэша:

1. Откройте консоль (`F12`)
2. Введите:
   ```javascript
   console.log("Tree.js version:", document.querySelector('script[src*="tree.js"]')?.src)
   ```
3. Должна быть **новая версия**

4. Попробуйте сохранить дерево
5. В консоли должно быть:
   ```
   [SAVE] ✅ Tree saved successfully
   ```

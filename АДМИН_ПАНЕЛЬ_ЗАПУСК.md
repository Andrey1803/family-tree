### Аутентификация
- `POST /api/auth/login` — вход пользователя
- `POST /api/auth/register` — регистрация

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/api/admin/stats` | GET | Статистика системы |
| `/api/admin/users` | GET | Список пользователей |
| `/api/admin/trees` | GET | Все деревья |
| `/api/admin/user/<id>/toggle` | POST | Блокировка/разблокировка |

## 🐛 Устранение проблем

### "Доступ запрещён" (403)
**Причина:** У пользователя нет флага `is_admin`

**Решение:**
```sql
UPDATE users SET is_admin = 1 WHERE login = 'Андрей Емельянов';
```

### "Неверный логин или пароль"
**Причина:** Пароль не совпадает с серверным

**Решение:**
1. Проверьте пароль в `Дерево/auth.py`
2. Используйте сброс пароля через админ-панель

### "Ошибка подключения к серверу"
**Причина:** Сервер Railway недоступен

**Решение:**
1. Проверьте интернет-соединение
2. Откройте https://ravishing-caring-production-3656.up.railway.app в браузере

### Админ-панель показывает только локальные данные
**Причина:** Не получен токен аутентификации

**Решение:**
1. Проверьте логи в `admin_panel_log.txt`
2. Убедитесь, что пароль введён правильно

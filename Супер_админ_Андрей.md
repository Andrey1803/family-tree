## 🔄 Развёртывание на сервере Railway

### Вариант 1: Автоматическое создание при деплое

При следующем деплое на Railway база данных будет автоматически обновлена:

1. Андрей Емельянов будет создан как супер-админ
2. Если пользователь уже существует — ему будут предоставлены права

### Вариант 2: Ручное выполнение на сервере

Подключитесь к серверу Railway и выполните:

```bash
# Подключение к SQLite на Railway
railway run sqlite3 /data/family_tree.db

# Создание пользователя (если не существует)
INSERT OR IGNORE INTO users (login, password_hash, email, is_admin)
VALUES ('Андрей Емельянов', '$sha256$хеш_пароля', 'andrey@familytree.local', 1);

# Предоставление прав (если пользователь уже есть)
UPDATE users SET is_admin = 1 WHERE login = 'Андрей Емельянов';

# Проверка
SELECT id, login, email, is_admin FROM users WHERE is_admin = 1;
```

### Вариант 3: Через API (если сервер запущен)

Используйте скрипт `make_andrey_super_admin.py`:

```bash
python make_andrey_super_admin.py
```

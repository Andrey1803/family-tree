# Исправления степени родства (05.03.2026)

## ✅ Что было исправлено

### 1. Двоюродные дяди/тёти
**Проблема:** Двоюродный дядя определялся как "Двоюродный дед"

**Причина:** В функции `_by_levels()` не было правильной обработки случая, когда:
- `person_level == 2` (дед person)
- `center_level == 3` (center на уровень ниже)

**Исправление:** Добавлена проверка для двоюродных дядь/тёть:
```python
elif person_level == 2 and center_level == 3:
    return "Двоюродный дядя" if gender == "Мужской" else "Двоюродная тётя"
```

### 2. Свойственники (тесть, тёща, шурин, золовка)
**Проблема:** Родственники супруга не определялись ("НЕ ОПРЕДЕЛЕНО")

**Причина:** Логика искала родство только через супруга person, но не через супруга center.

**Исправление:** Добавлена новая секция "3. СВОЙСТВЕННИКИ (через супруга center)":
- Проверяем, является ли person родителем супруга center → "Тесть"/"Теща"
- Проверяем, является ли person ребёнком родителей супруга → "Шурин"/"Золовка"

### 3. Внучатые племянники
**Проблема:** Сын племянницы определялся как "Внук 3-го колена"

**Причина:** Для боковой линии (через брата/сестру) использовалась терминология прямой линии.

**Исправление:**
- `person_level == 3` → "Внучатый племянник" / "Внучатая племянница"
- `person_level == 4` → "Внучатый племянник 2-го колена" / "Внучатая племянница 2-го колена"

### 4. Обновлён kinship_service.py
**Проблема:** Сервис использовал упрощённую логику родства.

**Исправление:** 
- Полностью переписана `_by_levels()` с той же логикой, что в базовом `kinship.py`
- Обновлён `_inlaw_term()` с правильными терминами для свойственников
- Добавлена логика для свойственников через супруга center
- Добавлена логика для внучатых племянников

## 🔧 Как работает алгоритм

1. **Прямые связи** - родители, дети, супруги
2. **Кровное родство** - через общих предков (уровни до предка)
3. **Свойственники (через супруга center)** - родители и сиблинги супруга
4. **Свойственники (через супруга person)** - остальные свойственники

### Уровни до общего предка:
- level=1: родитель/ребёнок
- level=2: дед/внук  
- level=3: прадед/правнук

### Формула для двоюродного родства:
- Общий предок + одинаковые уровни = двоюродные/троюродные братья
- Разница в 1 уровень = дядя/племянник (в т.ч. двоюродные)
- Разница в 2+ уровня = дед/внук (в т.ч. двоюродные)

# 👤 Персональная панель пользователя

## Что это?

**Персональная панель** — это веб-интерфейс для управления вашими данными семейного дерева, который:
- ✅ Подтягивает данные с сервера синхронизации
- ✅ Не вшита в код (данные загружаются динамически)
- ✅ Доступна всем авторизованным пользователям
- ✅ Показывает статистику и список деревьев

## Как открыть?

1. Запустите `Семейное_древо.exe`
2. Войдите под своим логином (например, `Андрей Емельянов`)
3. В меню выберите: **Файл → 👤 Моя панель**
4. Введите пароль (если не сохранён)
5. Панель загрузит данные с сервера

## Что показывает панель?

### 📊 Статистика (3 карточки)

| Карточка | Описание |
|----------|----------|
| 🌳 Деревья | Количество ваших деревьев на сервере |
| 👥 Персоны | Общее количество персон во всех деревьях |
| 🕐 Последняя активность | Дата последней синхронизации |

### 📁 Таблица деревьев

| Колонка | Описание |
|---------|----------|
| Название | Имя дерева |
| Обновлено | Дата последнего изменения |
| Размер | Размер файла дерева |
| Статус | ✅ Активно / ⚠️ Архив |

### 🔧 Доступные действия

- **📥 Скачать** — загрузить дерево на компьютер
- **📤 Загрузить** — отправить дерево на сервер
- **🗑️ Удалить** — удалить дерево с сервера

## Технические детали

### Сервер синхронизации

```
URL: https://ravishing-caring-production-3656.up.railway.app
API: /api/user/*
```

### API endpoints

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/api/user/stats` | GET | Статистика пользователя |
| `/api/user/trees` | GET | Список деревьев |
| `/api/user/trees/{id}` | DELETE | Удалить дерево |

### Файлы

| Файл | Описание |
|------|----------|
| `user_dashboard.py` | Персональная панель |
| `server_admin_dashboard.py` | Админ-панель |
| `auth.py` | Авторизация |
| `data/users.json` | Локальные пользователи |
| `data/login_remember.json` | Сохранённый логин/пароль |

## Безопасность

- ✅ Пароль запрашивается при первом входе
- ✅ Пароль сохраняется в `login_remember.json` (если выбрана галочка "Запомнить")
- ✅ Используется Basic Auth или Bearer Token для API
- ✅ Соединение по HTTPS

## Устранение проблем

### "Ошибка авторизации"
Проверьте логин/пароль. Если не помогает — зарегистрируйтесь на сервере.

### "Ошибка подключения"
Сервер синхронизации недоступен. Проверьте интернет-соединение.

### "Деревья не найдены"
У вас ещё нет деревьев на сервере. Создайте дерево и синхронизируйте.

# Файлы данных

Файлы создаются **автоматически** при первом запуске и хранятся в **корне проекта** (рядом с `main.py`) или **рядом с .exe** при запуске из собранного приложения.

## Расположение

| Режим | Папка данных |
|-------|--------------|
| Запуск `python main.py` | Корень проекта (папка с `main.py`) |
| Запуск `Семейное_древо.exe` | Папка с .exe |
| Веб-сервер | Корень проекта (родитель папки `web/`) |

## Файлы

| Файл | Описание |
|------|----------|
| `users.json` | Учётные записи (логин, хеш пароля) |
| `family_tree_Имя.json` | Дерево пользователя |
| `family_tree_Гость.json` | Дерево гостя |
| `user_settings_Имя.json` | Настройки пользователя |
| `login_remember.json` | Запомненный логин (desktop) |
| `palette.json` | Цветовая палитра |
| `window_settings.json` | Размер и позиция окна (desktop) |

Все эти файлы **исключены из репозитория** (см. `.gitignore`).

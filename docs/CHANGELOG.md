# История изменений «Семейное древо»

## [1.3.0] — 2026-02-22

### Добавлено
- ⏱️ **Временная шкала (Timeline)** — горизонтальная диаграмма жизней персон
  - Масштабирование (1, 2, 5, 10, 20 лет/см)
  - Фильтры (все, предки, с фото, мужчины, женщины)
  - Клик на полосу → переход к персоне
  - Горячая клавиша `Ctrl+T`

### Изменения
- 📦 **Сервисный слой** — выделена бизнес-логика в отдельные сервисы:
  - `TreeService` — операции с деревом
  - `ExportService` — экспорт в PDF, CSV
  - `BackupService` — резервное копирование
  - `UndoService` — отмена/повтор действий
  - `KinshipService` — вычисление родства
  - `ThemeService` — управление темами
  - `SettingsService` — настройки пользователя

- 🧪 **Тестирование** — добавлены юнит-тесты:
  - `test_services.py` — тесты сервисов
  - `test_models.py` — тесты моделей
  - `test_web.py` — тесты веб-API

- 🔒 **Безопасность (web)** — улучшена аутентификация:
  - Хеширование паролей через bcrypt
  - Обратная совместимость с SHA256
  - Генерация SECRET_KEY через secrets

- 📄 **Миграции данных** — автоматическое обновление формата JSON:
  - Версионирование файлов данных
  - Поддержка миграций 1.0.0 → 1.3.0

- 🐳 **Docker** — контейнеризация веб-версии

- 🚀 **CI/CD** — GitHub Actions:
  - Автотесты на Windows и Ubuntu
  - Проверка покрытия (coverage)
  - Сборка .exe при релизе
  - Публикация релизов

### Документация
- 📚 `docs/API.md` — документация веб-API и сервисов
- 📚 `pyproject.toml` — конфигурация pytest и coverage

## [1.1.0] — 2026-02-22

### Добавлено
- 💾 **Автосохранение** — каждые 5 минут при наличии изменений
- 📦 **Резервные копии** — до 10 последних версий
- ⌨️ **Горячие клавиши**:
  - `Ctrl+S` — сохранить
  - `Ctrl+Z` — отменить
  - `Ctrl+Y` — повторить
  - `Ctrl+F` — режим фокуса
  - `Ctrl+T` — временная шкала
- ⭐ **Индикатор изменений** — звёздочка в заголовке
- 🌓 **Тёмная тема** — переключение через меню
- ↩️ **Undo/Redo** — для операций редактирования

### Изменения
- Улучшена стабильность работы
- Оптимизирована производительность

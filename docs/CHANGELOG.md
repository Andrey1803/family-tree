# История изменений

## [1.0.0] — 2025-01 (первая версия)

### Desktop (tkinter)
- Построение семейного дерева с ортогональными линиями
- Карточки персон: имя, отчество, фамилия, даты, пол, фото
- Редактирование, добавление родственников (отец, мать, дети, братья, сёстры, супруги)
- Экспорт/импорт CSV
- Фильтры, режим фокуса, поиск
- Настраиваемая палитра цветов
- Регистрация протокола `derevo://` для открытия из веб-версии

### Веб (Flask)
- Авторизация: вход, регистрация, гость
- Просмотр и редактирование дерева
- Pan/zoom, контекстное меню
- Кнопки «Установить приложение» и «Открыть в приложении» (Windows desktop)
- Скачивание ZIP desktop-версии с веб-страницы

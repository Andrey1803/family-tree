# Сборка .exe для релиза

## Автоматическая сборка (GitHub Actions)

При публикации **релиза** на GitHub автоматически собирается `Семейное_древо.exe` и прикрепляется к релизу.

### Как создать релиз

1. Откройте https://github.com/Andrey1803/family-tree/releases
2. Нажмите **«Create a new release»**
3. Укажите тег, например `v1.0.0` (создайте новый тег)
4. Заголовок: «Семейное древо 1.0.0» (или другой)
5. Нажмите **«Publish release»**
6. Подождите 2–3 минуты — workflow соберёт .exe и прикрепит к релизу
7. Скачайте `Семейное_древо.exe` со страницы релиза

### Ручная сборка (без релиза)

1. Вкладка **Actions** → **Build Windows exe**
2. **Run workflow** → **Run workflow**
3. После завершения скачайте артефакт `Семейное_древо-exe` из результата

### Ссылка на .exe в веб-версии

На странице дерева (только Windows) отображается кнопка **«Скачать .exe»** — она ведёт на последний релиз.  
Репо по умолчанию: `Andrey1803/family-tree`. Можно переопределить переменной `GITHUB_REPO` (формат: `владелец/репозиторий`).

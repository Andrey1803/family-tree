# Сборка exe через GitHub (обход ESET)

Если сборка на вашем ПК падает из‑за антивируса, соберите exe в облаке:

1. Закоммитьте и запушьте код: `git add . && git commit -m "update" && git push`

2. На GitHub: вкладка **Actions** → выберите **Build Windows exe** → **Run workflow** → **Run workflow**

3. После завершения (2–5 мин) нажмите на выполненный workflow → внизу страницы скачайте артефакт **Tree** (zip с папкой Семейное_древо)

4. Распакуйте архив и запустите `Семейное_древо.exe` из папки
